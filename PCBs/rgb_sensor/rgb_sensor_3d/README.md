This directory contains 3D models used in this design

File [80216oplga4l.stp](./VEML3328/80216oplga4l.stp) is the original 3D model for the VEML3328 sensor that is on this board.
The file is included here without any changes and all of the rights belong to the sensor's manufacturer as shown in the included disclaimer [99917_read_me_3D_disclaimer.pdf](./VEML3328/99917_read_me_3D_disclaimer.pdf) and in the copyright notice shown lower.
Both of the files are direct extractions of [this](http://www.vishay.com/doc?80216) zip file which can be found linked from the manufacturer's [datasheet](https://www.vishay.com/docs/84968/veml3328.pdf).
